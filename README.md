# sizing_quick_analysis
scripts to quickly process and visualize data from aerosol sizing instruments

# MATLAB Scripts
*MATLAB version R2023b Update 10 (23.2.0.2859533)*
1. "cs_calculation.m" - Calculate the condensation sink given aerosol size distribution measurements (D and dN)
2. "sizing_stats.m" - Calculate key parameters of the size distribution given size distribution measurements (D and dN). Outputs include: 1) mode_D, 2) N_nucleation, 3) N_aitken, 4) N_accumulation, 5) N_submicron, 6) N_supermicron, 7) Dg, 8) sigma_g
